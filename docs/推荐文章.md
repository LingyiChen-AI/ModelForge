# ModelForge:小团队自己用的 NLP 模型平台

> 开源地址:<https://github.com/LingyiChen-AI/ModelForge> · 协议 Apache-2.0

说实话,很多 NLP 活儿真用不着大模型。情感识别、意图分类、实体识别、句子相似度这些,拿个小 BERT 微调一下,又快又便宜,还能本地离线跑,效果还更稳。

麻烦的是,自己从头搭一套「数据 → 训练 → 评估 → 上线」太碎了,每一步都得拼脚本。于是我做了 **ModelForge**,把这些揉进一个平台;顺手还加了条大模型 **Prompt 盲测评测**(这条线只评测、不微调大模型)。

先用一张表看看它都包了啥:

| 模块 | 它帮你干的事 | 一句话亮点 |
| --- | --- | --- |
| 数据集管理 | 上传即版本化,带 checksum | 训练随时能复现 |
| 模型训练 | 4 种 NLP 任务一键微调 | 实时进度 + MLflow 看曲线 |
| 模型测试 + 导出 | 测试集跑指标、出排行榜 | 每条预测一键导 Excel |
| 在线部署 | 满意版本一键上线 | 自动给接口 + curl 示例 |
| Badcase 闭环 | 线上坏例回收 → 标注 → 再训 | 自动算修复了多少条 |
| Prompt 评测 | 大模型提示词盲测 | 人工 + AI 双指标 |
| 权限与用户 | RBAC 多角色、API Key | 谁能干啥管得清 |

## 模型训练

四种任务都支持,选基础模型和数据集是联动的,不会选错;训练有实时进度,想看 loss 曲线点一下跳 MLflow。

| 任务类型 | 干啥用 | 典型场景 |
| --- | --- | --- |
| 文本分类 | 给一段文本打标签 | 情感识别、意图分类 |
| NER | 抽出文本里的实体 | 抽人名、地点、订单号 |
| 句对相似度 | 判断两句话像不像 | 问答匹配、去重 |
| 向量检索 | 把文本变成向量 | 语义搜索、召回 |
