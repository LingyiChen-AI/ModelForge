# ModelForge:小团队自己用的 NLP 模型平台

> 开源地址:<https://github.com/LingyiChen-AI/ModelForge>

说实话,很多 NLP 活儿真用不着大模型。情感识别、意图分类、实体识别、句子相似度这些,拿个小 BERT 微调一下,又快又便宜,还能本地离线跑,效果还更稳。

麻烦的是,自己从头搭一套「数据 → 训练 → 评估 → 上线」太碎了,每一步都得拼脚本。于是我做了 **ModelForge**,把这些揉进一个平台;顺手还加了个大模型 **Prompt 盲测评测**(这条线只评测、不微调大模型)。

下面挨个看看它能干啥。

![架构](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-architecture.png)

## 数据集管理

传数据自动存成版本快照,带 checksum,训练随时能复现。CSV / JSONL / Excel 都能导,还能直接下模板。

![数据集](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-datasets.png)

## 模型训练

四种任务都支持:文本分类、NER、句对相似度、向量检索。选基础模型和数据集是联动的,不会选错;训练有实时进度,想看 loss 曲线点一下跳 MLflow。

![训练](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-train.png)

## 模型测试 + 导出

跑一遍测试集自动出指标,同一个测试集横着比就是排行榜。每条预测还能一键导成 Excel,对错一目了然。

![模型测试](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-model-test.png)

![导出表格](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-export.png)

## 在线部署

满意的版本一键上线,自动给你 `/predict`、`/embed`、`/similarity` 接口,还附 curl 示例,复制就能调。

![部署](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-deploy.png)

## Badcase 闭环

线上踩的坑回传进来,在标注台标一标,一键生成训练集再训;再训的时候自动算修复了多少条。

![Badcase](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-badcase.png)

## Prompt 评测(我自己最爱用的)

接上大模型,把提示词做成带版本的模板,然后盲测——界面只给 A / B / C、藏掉版本,免得你先入为主。除了自己评,还能让另一个大模型当裁判自动打分,人工 + AI 两套指标一起看。

![Prompt 评测列表](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-prompt-eval-list.png)

![盲测工作台](https://cdn.jsdelivr.net/gh/LingyiChen-AI/images/ModelForge/images/2026/06/modelforge-prompt-blind-eval.png)

## 技术栈

FastAPI + Celery + React,底层 PostgreSQL / Redis / MinIO / MLflow,Apache-2.0 开源。
